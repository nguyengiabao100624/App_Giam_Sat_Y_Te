# Screenshots / Ảnh chụp màn hình

Place your app screenshots here with the following filenames:

| Filename | Description |
|----------|-------------|
| `banner.png` | Project banner (1280x640 recommended) |
| `login.png` | Login/Register screen |
| `home.png` | Home dashboard with vital signs |
| `chat.png` | AI medical assistant chat |
| `map.png` | Hospital finder map |
| `metric_detail.png` | Metric detail with charts |
| `notifications.png` | Notification history |
| `account.png` | Account/Profile screen |
| `pdf_report.png` | PDF report preview |

## Tips for great screenshots:

1. Use a clean device or emulator (hide status bar clutter)
2. Use **Android Studio Device Frame** tool to add phone frame
3. Recommended resolution: **1080x1920** (portrait)
4. For banner: Create a composite image with multiple screens
